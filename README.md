# HTML Simulators

Interactive Digital Technical-Scientific Products for Remote Sensing and Environmental Analysis

---

## Access

The simulators are publicly available at:

https://LGPierry.github.io/HTML-Simulators/

---

## Description

This repository contains a set of interactive HTML-based simulators developed within the Laboratory of Geotechnologies (LABGEOTEC) at the Federal University of Santa Maria (UFSM).

The simulators are designed to support applications in remote sensing, environmental analysis, and aquatic systems modeling, with an emphasis on conceptual understanding and applied interpretation.

All tools are web-based and can be accessed directly through a standard web browser without the need for installation.

---

## Available Simulators

Optical Competition Simulator
Simulates optical interactions between water constituents and their influence on spectral responses.

Multivariate Spectral Simulator
Applies multivariate approaches to model spectral behavior in aquatic environments.

Algal Bloom Simulator
Simulates bloom dynamics based on environmental and limnological variables.

Trophic State Simulator
Classifies trophic conditions using key limnological parameters.

Orbital Simulator – Sentinel-2
Models orbital parameters and acquisition geometry associated with the Sentinel-2 MSI sensor.
